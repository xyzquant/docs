To maintain consistency with XYZ’s linear contract structure, which is margined and settled exclusively in USDC, the system automatically converts these quotes into USD pricing. This ensures a seamless trading experience and eliminates currency friction for users. #### \*\*Data Source\*\* See the \[Specification Index\](/consolidated-resources/specification-index.md) page for pricing referencing sources. --- # Agent Instructions This documentation is published with GitBook.
